HTML and CSS only tabs

May 15, 2022 阅读时间：1 分钟
themes • syntax
markdown • css • html

Basic shortcodes 链接到标题

The following content:

{{< tabgroup >}}
{{< tab name="Hello" >}}
Hello World!
{{< /tab >}}

{{< tab name="Goodbye" >}}
Goodbye Everybody!
{{< /tab >}}
{{< /tabgroup >}}

Will generate:

Hello World!
Goodbye Everybody!

Right alighment 链接到标题

You can also align the tabs to the right:

{{< tabgroup align="right" >}}
{{< tab name="Hello" >}}
Hello World!
{{< /tab >}}

{{< tab name="Goodbye" >}}
Goodbye Everybody!
{{< /tab >}}
{{< /tabgroup >}}

Giving you this look:

Hello World!
Goodbye Everybody!

Markdown content 链接到标题

Any valid markdown can be used inside the tab:

{{< tabgroup align="right" style="code" >}}
{{< tab name="Ruby" >}}

```ruby
puts 'Hello'
```

{{< /tab >}}
{{< tab name="Python" >}}

```python
print('Hello')
```

{{< /tab >}}
{{< tab name="JavaScript" >}}

```js
console.log("Hello");
```

{{< /tab >}}
{{< /tabgroup >}}

And you get this lovely content:

In this case style="code" makes it look a little nicer for scenarios where your content purely a code block.
